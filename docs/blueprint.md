# **App Name**: Convico Landing

## Core Features:

- Email/Mobile Input: Allow users to enter their email address or mobile number.
- Password Input: Provide a secure password input field.
- Authentication: Verify user credentials against stored data.
- Landing Page Design: Visually appealing layout with Convico name and tagline.

## Style Guidelines:

- Primary color: Deep blue (#294380) to convey trust and professionalism.
- Background color: Very light blue (#EBF0F5), almost white, for a clean look.
- Accent color: Soft orange (#D97706) for CTAs to add warmth and contrast.
- Body and headline font: 'Inter' sans-serif font for a modern, neutral feel.
- Clean, minimal layout to highlight the sign-in form. The app name should be displayed in larger text at the top followed by the tagline 'AI conversations that connect'.
- Subtle transition animations when fields are in focus.
